Pour accéder à la carte:
- ssh udooer@"adresse_de_la_carte"

Si vous n'avez pas Python et pip, vous pouvez les installer en utilisant les commandes suivantes:
- sudo apt-get update
- sudo apt-get install python3
- sudo apt-get install python3-pip

Sinon installez Flask sur votre carte Udoo Neo avec la commande suivante:
- pip3 install flask

Pour lancer notre projet vous trouverez un script de lancement à éxecuter:
- ./lancement
Ce script lance aussi un premier test.

Pour voir les résultats vous pouvez accéder au site site web avec l'adresse suivante:
- https:/"adresse_de_la_carte":8888
